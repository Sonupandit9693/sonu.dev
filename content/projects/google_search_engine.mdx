# 🔎 Google Search Engine Clone

A modern Google Search Engine clone built with **React**, **Tailwind CSS**, and the **Google Custom Search API**. This project demonstrates a clean UI, responsive design, and integration of real search results—making it a great addition to your portfolio or resume.

---

## 🚀 Features

- 🔍 **Real-time Search**: Search the web using the official Google Custom Search API  
- 🖼️ **Image Search**: Toggle between web and image results  
- 📄 **Paginated Results**: Navigate results with intuitive pagination  
- 📱 **Responsive Design**: Works great on mobile and desktop  
- 🌙 **Modern UI**: Clean Google-like interface with Tailwind CSS  
- ⚡ **Quick Links**: Easy access to YouTube, GitHub, Facebook, Twitter, LinkedIn  
- 🏷️ **Category Tabs**: UI ready for All, Images, News, Videos, and Shopping  
- 🔗 **Live Links**: Results include clickable links, titles, and descriptions  

---

## 🛠️ Tech Stack

- **Framework**: React (with React Router)  
- **Bundler**: Vite (for fast builds)  
- **CSS**: Tailwind CSS  
- **API**: Google Custom Search API  
- **HTTP Client**: Axios  
- **Icons**: React Icons  
- **Tooling**: PostCSS & Autoprefixer
