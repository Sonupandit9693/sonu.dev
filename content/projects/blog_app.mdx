# 📝 Blogs.coding.tech

Welcome to **Blogs.coding.tech**! This repository powers my personal blog and serves as a portfolio of my **technical writing**, **coding tutorials**, and **projects**. Designed with both learning and sharing in mind, it features articles on **programming**, **development tips**, **open-source contributions**, and more.

---

## 🚀 Features

- ✍️ **Technical Blog Posts**: In-depth articles on software development, coding best practices, and technology insights  
- 🧠 **Tutorials**: Step-by-step guides and how-tos for various languages and frameworks  
- 💻 **Project Showcases**: Highlights of personal and collaborative projects with explanations  
- 📱 **Responsive Design**: Clean, modern, and mobile-friendly UI  
- 🌐 **Open Source**: Contributions and suggestions are welcome  

---

## 🛠️ Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS  
- **Content Management**: Markdown, MDX  
- **Hosting**: Vercel  
- **Other Tools**: Contentlayer, Framer Motion, Shiki
