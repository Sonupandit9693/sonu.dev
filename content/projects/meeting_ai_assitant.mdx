# 🤖 GenAI Meeting Assistant

**GenAI Meeting Assistant** is a full-stack, AI-driven platform designed to automate meeting tasks using **real-time transcription**, **summarization**, and **intelligent action extraction**. It's built for teams looking to streamline their post-meeting workflows and knowledge management.

---

## 🚀 Features

- 🎙️ **Real-time Transcription** using OpenAI Whisper
- 🧠 **Contextual Summarization** via GPT-4 for actionable insights
- 🔐 **JWT-secured APIs** for meeting and user management
- 🔄 **Automated Task Generation** and Slack follow-ups
- 🧪 **85%+ test coverage** using PyTest
- 📈 Phase 2 (ongoing): WebSocket live updates, analytics dashboard, calendar/Zoom integrations

---

## 🛠️ Tech Stack

- **Backend**: FastAPI, SQLAlchemy, Alembic, Celery, Redis, PyTest
- **Frontend**: Next.js 15, React, Tailwind CSS
- **AI/LLM**: GPT-4, Whisper API
- **Infra**: Docker Compose, GitHub Actions
