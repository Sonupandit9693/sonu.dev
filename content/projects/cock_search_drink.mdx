# 🍸 Cock_Search_Drink

**Cock_Search_Drink** is a simple yet powerful cocktail discovery app that allows users to search for cocktail recipes by name or ingredient. Whether you're a mixology enthusiast or just curious, this app makes it easy to find and follow recipes.

---

## 🔍 Features

- **Search Cocktails**: Find recipes by drink name or ingredient  
- **Detailed Recipes**: See ingredients, measurements, and preparation steps  
- **User-Friendly Interface**: Clean layout for intuitive use  
- **Mixology for All**: Perfect for beginners and pros alike  

---

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript  
- **API**: [TheCocktailDB](https://www.thecocktaildb.com/)  
- **Design**: Responsive across desktop and mobile
